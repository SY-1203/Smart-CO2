Smart CO2 is an AI powered Application that allows users to enter their daily practices after which the python code calculate and display the user's carbon footprint. It will also suggest alternate practices to the user

The project aims to increase awareness and create environmetally concious individuals.

The featurs of the app include:
1. Enter practices - through the input function
2. Calculate carbon footprint - calculation based on the UK Government GHG Conversion Factors
3. Display of charts - matplotlib module
4. Provide alternatives - AI answered

The program uses information and numbers from UK givernment website to ensure accuracy
