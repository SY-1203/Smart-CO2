Smart CO2 is an AI powered Application that allows users to select their practices from a pre-defined list of activities through which it uses python code to calculate and display the user's carbon footprint. It will also suggest alternate practices to the user

The project aims to increase awareness and create environmetally concious individuals.

The featurs of the app include:
1. Select practices
2. Calculate carbon footprint
3. Display of charts
4. Provide alternatives
